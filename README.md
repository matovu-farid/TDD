## Getting Started

To get the content of this project locally you need to run this command in your terminal:
``` bash
git clone your https://github.com/matovu-farid/TDD.git
cd TDD
bundle install
ruby main.rb
```

To run tests:
```
gem install rspec
rspec spec
```
